# Luna Demo Assets

These files are optional sample inputs for recording or judging.

Recommended flows:

## Attachments → Job Mission

Import:

- `demo_user_resume.docx`
- `job_description.txt`
- `portfolio_notes.md`

Then run **Job Mission**.

## Attachments → Artifact Studio

Import:

- `local_ai_research.pdf`
- `portfolio_notes.md`

Then run **Artifact Studio**.

## Mission Hub

Use:

- `meeting_transcript.txt`
- `invoice_demo_electronics.txt`

The seeded Mission Hub also works without importing these.

## Luna Lens OCR

Import:

- `luna_ocr_demo.png`

Then click **Explain this**.
