# Luna Submission Checklist

## Required

- [ ] Windows executable or installer built on Windows using `npm run dist`
- [ ] Demo video, 3–5 minutes
- [ ] App tested from a clean install
- [ ] Ollama optional setup tested
- [ ] Fallback mode tested without Ollama
- [ ] SmartScreen unsigned-app note included

## Demo video must show

- [ ] Local/privacy proof strip
- [ ] Judge Showcase
- [ ] Generated PDF/DOCX/PPTX/ZIP artifacts
- [ ] File automation preview + undo
- [ ] Knowledge Vault answer with evidence
- [ ] Luna Skill Creator or Mission Hub
- [ ] Trust Center audit log/export
- [ ] Luna Orb / command palette

## Before final recording

- [ ] Run `Reset demo`
- [ ] Pull models if using Ollama:
  - [ ] `qwen2.5:3b`
  - [ ] `nomic-embed-text`
- [ ] Close unrelated apps
- [ ] Prepare backup recording
- [ ] Practice fallback explanation

## Package contents

- [ ] Luna installer or portable executable
- [ ] Demo video
- [ ] Optional install guide PDF/Markdown
- [ ] Optional screenshots
- [ ] Optional feature matrix

## Telegram submission

- [ ] Upload final executable/installer or download link
- [ ] Upload demo video or video link
- [ ] Include short product pitch
- [ ] Include note: source code not required but available if asked

## Short pitch

Luna is a private local AI operating layer for the desktop. It combines local model inference, memory, knowledge retrieval, artifact generation, voice/command control, reversible automation, skill creation, desktop context understanding, and privacy forensics into one runnable desktop app.
