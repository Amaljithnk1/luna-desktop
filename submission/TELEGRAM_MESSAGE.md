# Telegram Submission Message

Hi team, submitting **Luna** for the desktop AI assistant hackathon.

**Luna** is a private local AI operating layer for the desktop. It runs with local Ollama models where available, falls back gracefully when models are missing, and combines conversation, memory, document understanding, artifact generation, reversible automation, desktop context, voice/command control, skill creation, and privacy forensics.

Key highlights:

- Local-first AI with Ollama + fallback mode
- Always-on-top Luna Orb and command palette
- Guided Judge Showcase mode
- Job Application, Research-to-Presentation, Meeting Notes, Invoice, Study Pack and Codebase missions
- PDF/DOCX/PPTX/HTML/CSV/JSON/ICS/ZIP exports
- Knowledge Vault with embedding retrieval when available
- Personal memory and adaptive context
- File automation with preview and full undo
- Luna Skill Creator for reusable local workflows
- Luna Lens for bounded desktop/screenshot understanding
- Trust Center with audit logs, network counter, SQLite status, data export and reset
