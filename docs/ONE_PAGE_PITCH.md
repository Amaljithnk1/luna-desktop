# Luna — One Page Pitch

## One-liner

**Luna is a private local AI operating layer for the desktop.**

It combines local AI, memory, document understanding, artifact generation, safe automation, voice/command control, skill creation, desktop context, and privacy forensics into one runnable desktop assistant.

---

## Why it matters

Most AI assistants are cloud chatboxes. They answer questions, but they do not deeply understand the user's local workspace, safely act on files, create polished outputs, or prove what data they touched.

Luna is designed around a different idea:

```txt
Observe → Understand → Act → Explain → Remember → Stay under user control
```

---

## Strongest proof moments

- **Local-first proof:** external request counter, resource meter, Ollama/fallback status.
- **Artifacts, not chat only:** PDF, DOCX, PPTX, HTML, Markdown, CSV, JSON, ICS, ZIP.
- **Safe automation:** preview → permission → execute → manifest → full undo.
- **AI Presenter:** Luna can narrate and run its own demo.
- **Luna Skill Creator:** turns plain-English workflows into reusable local skills.
- **Trust Center:** audit log, SQLite status, data export, reset controls.
