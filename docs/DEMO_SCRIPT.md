# Luna 5-Minute Demo Script

## Core line

**Luna is a private local AI operating layer for your desktop — it can understand context, create artifacts, automate safely, explain what it did, and keep the user in control.**

---

## 0:00–0:25 — Hook: local-first proof

Open Luna. Complete/skip onboarding.

Say:

> Luna is not another cloud chatbot. It is a local AI operating layer for the desktop.

Show the top proof strip:

- Ollama connected / fallback ready
- External requests counter
- CPU/RAM usage

Optional if stable: turn Wi-Fi off before starting.

---

## 0:25–1:15 — Judge Showcase

Go to **Judge Showcase** and click **Run full showcase**.

Say:

> This guided mode runs Luna’s strongest end-to-end proof path: job artifacts, presentation generation, evidence search, reversible automation, skill creation, and adaptive memory.

Let the timeline run and point out generated artifacts.

---

## 1:15–2:00 — Artifact proof

Open **Generated Artifacts** from the showcase or go to **Artifact Studio**.

Reveal:

- `luna_research_deck.pptx`
- `research_brief.pdf`
- `research_presentation_package.zip`

Say:

> Luna turns local documents into usable files, not just chat text.

---

## 2:00–2:45 — Safe automation with undo

Go to **Automation**.

Click:

1. Analyze demo Downloads
2. Approve and run
3. Undo entire mission

Say:

> Luna does not blindly modify the desktop. It previews actions, asks approval, writes a manifest, and can undo the full mission.

---

## 2:45–3:25 — Knowledge Vault + Memory

Go to **Knowledge Vault**.

Click **Index demo docs**, then ask:

```txt
What does Luna prove about privacy and safe automation?
```

Show evidence cards.

Then go to **Memory** and show relevant memory/adaptive context.

Say:

> Luna retrieves only relevant local memory and document evidence instead of dumping everything into the model.

---

## 3:25–4:05 — Luna Skill Creator / Mission Hub

Go to **Luna Skill Creator** or **Mission Hub**.

Run one:

- Invoice Extractor
- Meeting Notes
- Study Pack
- Codebase Explainer

Say:

> If another workflow is needed, Luna can turn it into a reusable local skill with inputs, permissions, steps, and outputs.

---

## 4:05–4:35 — Luna Lens / Voice / Orb

Click the floating orb or press `Ctrl/Cmd + Shift + L`.

Run a command:

```txt
Create a presentation from my local research notes
```

Optionally show **Voice** transcript mode.

Say:

> Luna can be accessed like a desktop command layer — through the orb, shortcut, command palette, or voice transcript mode.

---

## 4:35–5:00 — Trust Center close

Go to **Trust Center**.

Show:

- Audit summary
- Event log
- SQLite database status
- Export trust data
- Delete/reset local data

Close with:

> Everything you saw ran through Luna’s local-first pipeline: local model or transparent fallback, local files, local artifacts, audit logs, privacy export, and safe undo. Luna is a private AI operating layer for the desktop.
